# 🎥 AI YouTube Video Analyzer

An AI-powered web application that analyzes YouTube videos and generates structured insights using Large Language Models. Built with **Python**, **Streamlit**, **Agno**, and **Groq**, this project transforms educational and technical videos into concise summaries, timestamps, study notes, and interview-focused learning material.

---

## ✨ Features

* 🎥 Analyze any public YouTube video
* 📋 Generate detailed AI-powered summaries
* ⏱️ Create meaningful timestamps for major topics
* 📚 Extract key learning points
* 💡 Explain technical concepts in simple English
* 💻 Identify technologies, libraries, frameworks, and tools mentioned
* 🎯 Generate interview questions with answers
* 📝 Produce structured study notes
* 📥 Download the analysis report
* 🎨 Clean and responsive Streamlit interface

## 🚀 Installation

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/AI-Youtube-Video-Analyzer.git
cd AI-Youtube-Video-Analyzer
```

---

### 2. Create a Virtual Environment

**Windows**

```bash
python -m venv venv
venv\Scripts\activate
```

**Linux / macOS**

```bash
python3 -m venv venv
source venv/bin/activate
```

---

### 3. Install Dependencies

```bash
pip install -r requirements.txt
```

---

### 4. Configure Environment Variables

Create a `.env` file in the project root.

```env
GROQ_API_KEY=your_groq_api_key
```

---

### 5. Run the Application

```bash
streamlit run app.py
```

The application will be available at:

```
http://localhost:8501
```
